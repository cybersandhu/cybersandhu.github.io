# Cyber_Sandhu — Static Site

This repo hosts my personal site on **GitHub Pages** at `https://cybersandhu.github.io`.

## Structure
- `index.html` — single-page site
- `styles.css` — modern dark-first styling
- `script.js` — theme toggle + smooth scrolling
- `assets/` — logo, hero artwork, favicon

## Local preview
Open `index.html` in a browser, or run a simple server:
```bash
python3 -m http.server 8080
```

## Deploy
Commit + push to `main`. GitHub Pages serves from `main` (root).
